Daily News by NanoClaw

A NanoClaw Agent running on my homeserver delivering daily news summary from multiple sources every morning. The agent updates the files in the /data folder of the Git repository every morning, commits and pushes it to the repository, and Inkgit🫟 builds this webpage.
